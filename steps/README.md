## Как пользоваться

1. Открой [визуальную роадмапу](../agent-roadmap.html) — обзор и детали этапов.
2. Иди по папкам **по порядку**. Не перескакивай: каждый шаг опирается на предыдущий.
3. На этапе 1: [промпты](01-toy-project/prompts/) + [базовые правила](01-toy-project/prompts/00-base-rules.md) для AI.
4. На каждом этапе сверяйся с **edge-cases.md** в папке шага.
5. Сдавай этап по **DOD.md** (где есть) с ментором.
